AI Hub - TurboWarp AI 扩展插件
================================

这个文件夹包含一个 TurboWarp 扩展插件，让你的 Scratch 项目可以接入 AI 能力。

【文件说明】
- ai-hub.js          扩展主文件（在 TurboWarp 里导入这个）
- AI-HUB-USAGE.md    详细使用文档
- SPEC.md            技术规格说明

【支持的功能】
1. AI 聊天对话（DeepSeek / Qwen）
   - 创建多个 AI NPC 角色
   - 角色会记住对话历史
   - 支持同步/异步两种对话模式

2. 图片生成（Pollinations / Qwen）
   - 生成图片并自动添加到角色造型
   - 生成图片并自动添加到舞台背景
   - Pollinations 完全免费，无需 API Key，无 CORS 问题，强烈推荐！

3. 图片识别（Qwen）
   - 识别/分析图片内容
   - 不受当前 provider 影响，始终使用 Qwen

【如何使用】
1. 打开 TurboWarp 编辑器 (https://turbowarp.org/editor)
2. 点击左下角「添加扩展」→「导入插件」
3. 选择 ai-hub.js 文件
4. 在积木栏找到「AI Hub」分类即可使用

【需要准备】
- DeepSeek API Key（聊天）：https://platform.deepseek.com/
- Qwen API Key（图片识别）：https://bailian.console.aliyun.com/
- Pollinations（图片生成）：完全免费，不需要 API Key！
