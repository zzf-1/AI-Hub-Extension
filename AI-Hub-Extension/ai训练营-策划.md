# 🏕️ AI 训练营 — Scratch 项目策划文档

## 一、项目概览

| 项目名 | AI 训练营 |
|--------|-----------|
| 类型 | 教育类互动小游戏 |
| 目标用户 | 中学生（12-15岁） |
| 学习目标 | 了解 AI 是什么、会写提示词、知道 AI 的局限 |
| 平台 | TurboWarp + AI Hub 扩展 |
| 时长 | 10-15 分钟 |

## 二、场景设计

### 场景 1：训练营大厅
- 背景：科技感大厅，3 扇门分别通往 3 个关卡
- 显示玩家名字、已通关数

### 场景 2：第 1 关 — 初识 AI（小智的房间）
- 背景：明亮温馨的书房
- NPC：小智（年轻女导师）

### 场景 3：第 2 关 — 提示词工程（老严的房间）
- 背景：简洁的实验室
- NPC：老严（中年男导师，严肃）

### 场景 4：第 3 关 — AI 的极限（博博士的房间）
- 背景：高科技实验室
- NPC：博博士（学者型导师）

### 场景 5：毕业典礼
- 背景：礼堂，3 个导师一起出场

## 四、关卡详细设计

### 第 1 关：初识 AI

```
游戏流程：
1. 小智自我介绍，问玩家名字
2. 小智出 3 道关于 AI 的知识题
   玩家用自然语言回答（不是选择）
   小智评判答案并给出解释
3. 至少答对 2 题 → 通关

题目示例：
- "你觉得AI是什么意思？用你自己的话说说看。"
- "你身边有哪些东西用到了AI？举例说说。"
- "AI和普通程序有什么不同？"
```

### 第 2 关：提示词工程

```
游戏流程：
1. 老严讲什么是"提示词"
   老严："提示词就是你和AI说的话。说得越清楚，AI就越懂你。"

2. 挑战1：让它说"蓝色"
   规则：写提示词让AI说出"蓝色"，但不能包含"蓝"字
   老严评判结果

3. 挑战2：让它推荐书
   规则：分别用模糊和清晰的提示词让AI推荐书
   老严对比两种回答的差距

4. 挑战3：自由创作
   玩家自己出题，老严评价提示词质量

通关条件：完成 3 个挑战
```

### 第 3 关：AI 的极限

```
游戏流程：
博博士给 3 个场景，玩家判断"AI能做"还是"AI不能做"

场景1：AI 算错了数学题
→ 揭示：AI 不擅长精确计算

场景2：AI 不知道昨天发生了什么
→ 揭示：AI 知识有截止日期

场景3：AI 不理解复杂情感
→ 揭示：AI 没有真实体验过世界

通关条件：3 题全答对
```
